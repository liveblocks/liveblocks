These examples estimate monthly bills for four common products. Each section
shows the scenario, metered usage, and how [plan credits](/docs/pricing/plans)
apply.

For current unit prices, see
[Pro metered usage](/docs/pricing/plans/pro#Metered-usage) and
[Team metered usage](/docs/pricing/plans/team#Metered-usage). Track your own
usage on the [dashboard usage page](/dashboard/usage).

## How billing works

Every paid plan includes a monthly credit allowance. Credits are applied to
metered usage first. You only pay overage when usage exceeds your included
credits.

| Term             | Meaning                                           |
| ---------------- | ------------------------------------------------- |
| Credits used     | Total metered usage cost for the month            |
| Plan base        | Monthly plan price                                |
| Included credits | Credits included with your plan                   |
| Overage          | Credits used minus included credits (if positive) |
| **Total**        | Plan base + overage                               |

Team plans can include more credits than the plan costs. That bonus reduces your
bill when usage stays within the allowance.

## Document editor

A collaborative editor where people and AI edit the same document together,
synced and versioned through
[Storage](/docs/collaboration-features/multiplayer/sync-engine/liveblocks-storage).

This example uses the [Pro plan](/docs/pricing/plans/pro) at $30/mo.

### Scenario

- 500 monthly active users
- 100 editing daily
- ~10 min each per day

### Metered usage

| Item                                                                    | Calculation                                      | Total       |
| ----------------------------------------------------------------------- | ------------------------------------------------ | ----------- |
| [Realtime collaboration minutes](/docs/pricing/plans/pro#Metered-usage) | 100 editors × 10 min × 22 days                   | 22,000 mins |
| Collaboration cost                                                      | 22,000 × $0.002                                  | $44         |
| [Realtime data storage updates](/docs/pricing/plans/pro#Metered-usage)  | <span className="text-product-disabled">–</span> | 10,000,000  |
| Update cost                                                             | 10M × $0.000001                                  | $10         |
| [Realtime data stored](/docs/pricing/plans/pro#Metered-usage)           | <span className="text-product-disabled">–</span> | 5 GB        |
| Storage cost                                                            | 5 × $0.15                                        | $0.75       |
| **Credits used**                                                        | $44 + $10 + $0.75                                | **~$55**    |

Storage is cumulative and does not reset monthly. See
[platform limits](/docs/platform/limits) for other caps.

### Your bill

| Item             | Calculation                                      | Total       |
| ---------------- | ------------------------------------------------ | ----------- |
| Credits used     | <span className="text-product-disabled">–</span> | ~$55        |
| Plan base (Pro)  | <span className="text-product-disabled">–</span> | $30         |
| Included credits | <span className="text-product-disabled">–</span> | $30         |
| Overage          | $55 − $30                                        | ~$25        |
| **Total**        | $30 + $25                                        | **~$55/mo** |

Pro includes $30 of credits for $30. Usage above $30 is billed as overage.

## Notification inbox

Custom notifications are product events your app triggers, like a finished
export or a status change. Deliver them in-app or by email and Slack through
[Notifications](/docs/collaboration-features/notifications).

This example uses the [Team plan](/docs/pricing/plans/team) at $600/mo.

### Scenario

- 10,000 monthly active users
- ~8 custom notifications each per month

### Metered usage

| Item                                                                   | Calculation                                      | Total    |
| ---------------------------------------------------------------------- | ------------------------------------------------ | -------- |
| [Monthly custom notifications](/docs/pricing/plans/team#Metered-usage) | 10,000 users × 8                                 | 80,000   |
| Notification cost                                                      | 80,000 × $0.01                                   | $800     |
| **Credits used**                                                       | <span className="text-product-disabled">–</span> | **$800** |

[Collaboration notifications](/docs/collaboration-features/notifications/concepts#Collaboration-notifications)
(mentions, replies, thread updates) are separate and included free on paid
plans.

### Your bill

| Item             | Calculation                                      | Total       |
| ---------------- | ------------------------------------------------ | ----------- |
| Credits used     | <span className="text-product-disabled">–</span> | $800        |
| Plan base (Team) | <span className="text-product-disabled">–</span> | $600        |
| Included credits | <span className="text-product-disabled">–</span> | $600        |
| Overage          | $800 − $600                                      | $200        |
| **Total**        | $600 + $200                                      | **$800/mo** |

## Canvas

A shared visual canvas where collaborators make changes simultaneously and see
each other's cursors and selections through
[Presence](/docs/ready-made-features/presence) and
[Multiplayer](/docs/collaboration-features/multiplayer).

This example uses the [Team plan](/docs/pricing/plans/team) at $900/mo.

### Scenario

- 2,000 monthly active users
- 400 collaborating daily
- ~80 min each per day

### Metered usage

| Item                                                                     | Calculation                                      | Total        |
| ------------------------------------------------------------------------ | ------------------------------------------------ | ------------ |
| [Realtime collaboration minutes](/docs/pricing/plans/team#Metered-usage) | 400 daily × 80 min × 22 days                     | 704,000 mins |
| Session cost                                                             | 704,000 × $0.002                                 | $1,408       |
| **Credits used**                                                         | <span className="text-product-disabled">–</span> | **$1,408**   |

Collaboration minutes count active time in a room, billed per minute.

### Your bill

| Item             | Calculation                                      | Total         |
| ---------------- | ------------------------------------------------ | ------------- |
| Credits used     | <span className="text-product-disabled">–</span> | $1,408        |
| Plan base (Team) | <span className="text-product-disabled">–</span> | $900          |
| Included credits | <span className="text-product-disabled">–</span> | $1,000        |
| Overage          | $1,408 − $1,000                                  | $408          |
| **Total**        | $900 + $408                                      | **$1,308/mo** |

The Team $900 plan includes $1,000 in credits, $100 more than the plan price.
You only pay overage above $1,000.

## Comments

Comment threads where users leave feedback and reply in context through
[Comments](/docs/collaboration-features/comments).

This example uses the [Team plan](/docs/pricing/plans/team) at $1,500/mo.

### Scenario

- 2,000 monthly active users
- ~100 comments each per month

### Metered usage

| Item                                                       | Calculation                                      | Total      |
| ---------------------------------------------------------- | ------------------------------------------------ | ---------- |
| [Comments created](/docs/pricing/plans/team#Metered-usage) | 2,000 users × 100                                | 200,000    |
| Comment cost                                               | 200,000 × $0.01                                  | $2,000     |
| **Credits used**                                           | <span className="text-product-disabled">–</span> | **$2,000** |

Each comment posted in a room counts toward usage.

### Your bill

| Item             | Calculation                                      | Total         |
| ---------------- | ------------------------------------------------ | ------------- |
| Credits used     | <span className="text-product-disabled">–</span> | $2,000        |
| Plan base (Team) | <span className="text-product-disabled">–</span> | $1,500        |
| Included credits | <span className="text-product-disabled">–</span> | $1,750        |
| Overage          | $2,000 − $1,750                                  | $250          |
| **Total**        | $1,500 + $250                                    | **$1,750/mo** |

The Team $1,500 plan includes $1,750 in credits, $250 more than the plan price.
