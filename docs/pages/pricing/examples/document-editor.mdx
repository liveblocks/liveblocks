This example estimates a monthly bill for a collaborative document editor with
[Storage](/docs/collaboration-features/multiplayer/sync-engine/liveblocks-storage).
It assumes <strong>500 monthly active users</strong>, with <strong>100 editing
daily</strong> for <strong>10 minutes each</strong> (22 working days).

Recommended plan: [Pro](/docs/pricing/plans/pro) at $30/mo ($30 credits
included).

## What you’re billed for

- [Realtime collaboration minutes](/docs/pricing/plans/pro#Metered-usage):
  active editing time, billed at $0.002 per minute
- [Realtime data storage updates](/docs/pricing/plans/pro#Metered-usage): each
  change saved to the document, billed at $1 per million

Realtime data stored is cumulative and does not reset monthly. See
[Limits](/docs/platform/limits) for storage caps.

## Usage estimate

| Item                                                                    | Calculation                    | Total       |
| ----------------------------------------------------------------------- | ------------------------------ | ----------- |
| [Realtime collaboration minutes](/docs/pricing/plans/pro#Metered-usage) | 100 editors × 10 min × 22 days | 22,000 mins |
| Editing cost                                                            | 22,000 × $0.002                | $44         |
| [Realtime data storage updates](/docs/pricing/plans/pro#Metered-usage)  | avg per Pro storage customer   | 2,000,000   |
| Update cost                                                             | 2M @ $1 per 1M                 | $2          |
| **Total usage**                                                         | $44 + $2                       | **$46**     |

## Your bill

| Item                        | Billed monthly | Billed annually |
| --------------------------- | -------------- | --------------- |
| Pro plan ($30 credits)      | $30/mo         | $25/mo          |
| Usage beyond credits (PAYG) | $16            | $16             |
| **Total**                   | **$46/mo**     | **$41/mo**      |

Credits apply to usage first; overage is billed at the same rates. See
[Billing](/docs/pricing/billing) for how credits, plan base, and PAYG add up.
Billing annually saves 2 months on the plan price; overage is unchanged.
