A shared visual canvas where collaborators make changes simultaneously and see
each other's cursors and selections through
[Presence](/docs/ready-made-features/presence) and
[Multiplayer](/docs/collaboration-features/multiplayer).

This example uses the [Team plan](/docs/pricing/plans/team) at $900/mo.

## Scenario

- 2,000 monthly active users
- 400 collaborating daily
- ~80 min each per day

## Metered usage

| Item                                                                     | Calculation                                      | Total        |
| ------------------------------------------------------------------------ | ------------------------------------------------ | ------------ |
| [Realtime collaboration minutes](/docs/pricing/plans/team#Metered-usage) | 400 daily × 80 min × 22 days                     | 704,000 mins |
| Session cost                                                             | 704,000 × $0.002                                 | $1,408       |
| **Credits used**                                                         | <span className="text-product-disabled">–</span> | **$1,408**   |

Collaboration minutes count active time in a room, billed per minute.

## Your bill

| Item             | Calculation                                      | Total         |
| ---------------- | ------------------------------------------------ | ------------- |
| Credits used     | <span className="text-product-disabled">–</span> | $1,408        |
| Plan base (Team) | <span className="text-product-disabled">–</span> | $900          |
| Included credits | <span className="text-product-disabled">–</span> | $1,000        |
| Overage          | $1,408 − $1,000                                  | $408          |
| **Total**        | $900 + $408                                      | **$1,308/mo** |

The Team $900 plan includes $1,000 in credits, $100 more than the plan price.
You only pay overage above $1,000.
