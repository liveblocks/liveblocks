This example estimates a monthly bill for a custom notification inbox with
[Notifications](/docs/ready-made-features/notifications). It assumes
<strong>10,000 monthly active users</strong>, each receiving <strong>8 custom
notifications</strong> per month.

Recommended plan: [Team](/docs/pricing/plans/team) at $600/mo ($600 credits
included).

## What you're billed for

- [Monthly custom notifications](/docs/pricing/plans/team#Metered-usage): each
  event your app triggers, billed at $0.01 per notification.
- [Collaboration notifications](/docs/collaboration-features/notifications/concepts#Collaboration-notifications)
  (mentions, replies, thread updates) are separate and included free on paid
  plans. They are not counted here.

## Usage estimate

| Item                                                                   | Calculation      | Total    |
| ---------------------------------------------------------------------- | ---------------- | -------- |
| [Monthly custom notifications](/docs/pricing/plans/team#Metered-usage) | 10,000 users × 8 | 80,000   |
| Notification cost                                                      | 80,000 × $0.01   | $800     |
| **Total usage**                                                        |                  | **$800** |

## Your bill

| Item                        | Billed monthly | Billed annually |
| --------------------------- | -------------- | --------------- |
| Team plan ($600 credits)    | $600/mo        | $500/mo         |
| Usage beyond credits (PAYG) | $200           | $200            |
| **Total**                   | **$800/mo**    | **$700/mo**     |

Credits apply to usage first; overage is billed at the same rates. See
[Billing](/docs/pricing/billing) for how credits, plan base, and PAYG add up.
Billing annually saves 2 months on the plan price; overage is unchanged.
