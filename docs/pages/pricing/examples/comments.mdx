Comment threads where users leave feedback and reply in context through
[Comments](/docs/collaboration-features/comments).

This example uses the [Team plan](/docs/pricing/plans/team) at $1,500/mo.

## Scenario

- 2,000 monthly active users
- ~100 comments each per month

## Metered usage

| Item                                                       | Calculation                                      | Total      |
| ---------------------------------------------------------- | ------------------------------------------------ | ---------- |
| [Comments created](/docs/pricing/plans/team#Metered-usage) | 2,000 users × 100                                | 200,000    |
| Comment cost                                               | 200,000 × $0.01                                  | $2,000     |
| **Credits used**                                           | <span className="text-product-disabled">–</span> | **$2,000** |

Each comment posted in a room counts toward usage.

## Your bill

| Item             | Calculation                                      | Total         |
| ---------------- | ------------------------------------------------ | ------------- |
| Credits used     | <span className="text-product-disabled">–</span> | $2,000        |
| Plan base (Team) | <span className="text-product-disabled">–</span> | $1,500        |
| Included credits | <span className="text-product-disabled">–</span> | $1,750        |
| Overage          | $2,000 − $1,750                                  | $250          |
| **Total**        | $1,500 + $250                                    | **$1,750/mo** |

The Team $1,500 plan includes $1,750 in credits, $250 more than the plan price.
