This example estimates a monthly bill for in-context comment threads with
[Comments](/docs/collaboration-features/comments). It assumes <strong>2,000
monthly active users</strong>, each posting <strong>100 comments per
month</strong>.

Recommended plan: [Team](/docs/pricing/plans/team) at $1,500/mo ($1,750 credits
included).

## What you’re billed for

[Comments created](/docs/pricing/plans/team#Metered-usage): each comment posted
in a room, billed at $0.01 per comment.

## Usage estimate

| Item                                                       | Calculation       | Total      |
| ---------------------------------------------------------- | ----------------- | ---------- |
| [Comments created](/docs/pricing/plans/team#Metered-usage) | 2,000 users × 100 | 200,000    |
| Comment cost                                               | 200,000 × $0.01   | $2,000     |
| **Total usage**                                            |                   | **$2,000** |

## Your bill

| Item                        | Billed monthly | Billed annually |
| --------------------------- | -------------- | --------------- |
| Team plan ($1,750 credits)  | $1,500/mo      | $1,250/mo       |
| Usage beyond credits (PAYG) | $250           | $250            |
| **Total**                   | **$1,750/mo**  | **$1,500/mo**   |

Credits apply to usage first; overage is billed at the same rates. See
[Billing](/docs/pricing/billing) for how credits, plan base, and PAYG add up.
Billing annually saves 2 months on the plan price; overage is unchanged.

The $1,500/mo Team plan includes $1,750 in credits, $250 more than the plan
price. You only pay overage above that allowance.
