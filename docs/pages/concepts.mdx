## Storage

Storage represents the items people can interact with in the room. In the
physical world, storage could be represented as documents and notes on a
whiteboard.

</Markdown>

</Grid>

## Storage persistence

Storage data automatically persists when people leave the room. The data can
also be cleared and stored on your database using the API endpoints.
