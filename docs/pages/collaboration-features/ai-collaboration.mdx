Bring realtime multiplayer to your AI agents with Feeds and Presence. Connect
any workflow and use Liveblocks APIs to update your application state live with
UI updates, comment responses, AI presence, and more. It’s also possible to
create and update feeds when an AI assistant is
[mentioned in a comment](/docs/collaboration-features/comments/users-and-mentions),
or trigger a feed from a custom UI component.

Any workflows are compatible, such as n8n, LangChain, Crew AI, or custom setups.
Feeds is ideal for multi-agent orchestration, automation, and complex pipelines.

### n8n nodes

A set of Liveblocks nodes are available for [n8n](https://n8n.io/), which you
can use directly in your workflow. Each Liveblocks REST API endpoint has a
corresponding n8n node, allowing you to create rooms, add to feeds, update
multiplayer state, and more.
