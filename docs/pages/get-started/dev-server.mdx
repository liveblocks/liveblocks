Liveblocks is a realtime collaboration infrastructure for building performant
collaborative experiences. Follow the following steps to set up a local
development environment for building with Liveblocks.

<Banner title="Only Multiplayer is supported" type="warning">

The Liveblocks dev server only supports
[Storage](/ready-made-features/multiplayer/sync-engine/liveblocks-storage) and
[Yjs](/ready-made-features/multiplayer/sync-engine/liveblocks-yjs).
[Learn more](/docs/tools/dev-server).

</Banner>
