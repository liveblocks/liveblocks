The Liveblocks dev server is a Bun-based server that allows you to develop and
test multiplayer features locally without connecting to Liveblocks servers,
using our open-source `@liveblocks/server` package internally. It supports a
subset of Liveblocks features including
[Liveblocks Storage](/ready-made-features/multiplayer/sync-engine/liveblocks-storage)
and
[Liveblocks Yjs](/ready-made-features/multiplayer/sync-engine/liveblocks-yjs).
