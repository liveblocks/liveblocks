# liveblocks

Liveblocks command line interface.

## Run locally

```bash
docker run -p 1153:1153 ghcr.io/liveblocks/liveblocks/dev-server:latest
```

See [DOCKER.md](./DOCKER.md) for configuration, persisting room data, and more.
